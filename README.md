# Minecraft Serverlist

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Start frontend only

Start the development server on `http://localhost:3000`

```bash
npm run dev
```

## Start everything

Ensure you have Docker Desktop opened

Start the development server on `http://localhost:3000`

Start Grafana server on `http://localhot:3001`

```bash
docker compose up --build -d
```
